# GNHub
Roblox Script code: 
```loadstring(game:HttpGet("https://raw.githubusercontent.com/nici002018/GNHub/master/GNHub.lua"))()```

## Supported gemas
1. War Tycoon
2. Arsenal

## Universal
1. ESP for all games
2. Speed boost for all games
3. Jump boost for all games
4. Aimbot for all games
